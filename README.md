Запуск приолжения в режиме отладки:
1. cd admin-panel
2. npm install
3. npm run dev

Как собрать приложение для Production:
vite build

Используемые пакеты и их установка:
1. react router dom ^6.11.2 (npm i react-router-dom)
2. scss (npm i --save-dev sass)
3. axios (npm i axios)
4. npm i react-redux
